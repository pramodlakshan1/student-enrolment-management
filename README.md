# Student Course Enrollment Management System

## Overview
Full-stack web app for managing student enrollments. Built with React (frontend), Node.js/Express (backend), MongoDB.

## Setup
1. Clone repo.
2. Backend: `cd backend`, `npm install`, update .env, `node server.js`
3. Frontend: `cd frontend`, `npm install`, `npm run dev`

## Features
- Dashboard with student list, search, filter, pagination.
- Add/edit forms with validation.
- REST API: GET/POST/PUT/DELETE.
- Pending status highlighting.
